# Brainstorming de Diseño: Sierra Guard - Control de Plagas Segovia

## Contexto
Página web de alto rendimiento para empresa de control de plagas con enfoque en CRO (Conversion Rate Optimization), Mobile-First y SEO local. Objetivo: captar clientes en Segovia con diseño persuasivo y arquitectura de silos geográficos.

---

## <response>

### Idea 1: Minimalismo Técnico + Autoridad Profesional
**Probabilidad: 0.08**

#### Design Movement
**Brutalism Digital + Modernismo Corporativo**

#### Core Principles
1. **Autoridad mediante claridad**: Tipografía serif de alto contraste + espacios amplios = confianza profesional
2. **Funcionalidad extrema**: Cada elemento tiene propósito; nada decorativo
3. **Velocidad visual**: Estructura jerárquica clara que guía al usuario hacia CTA en máximo 2 segundos
4. **Rigor técnico**: Grillas asimétricas, alineaciones precisas, proporciones matemáticas

#### Color Philosophy
- **Paleta**: Gris oscuro (#1a1a1a) + Verde forestal profundo (#1d4620) + Blanco puro
- **Razonamiento**: Verde evoca naturaleza/salud (antídoto a plagas); gris transmite profesionalismo industrial; alto contraste = accesibilidad + urgencia
- **Intención emocional**: "Somos científicos, no improvisamos"

#### Layout Paradigm
- **Hero asimétrico**: Texto grande a la izquierda (70% ancho), imagen técnica/microscópica a la derecha
- **Secciones en bloques verticales alternados**: Texto-Imagen-Texto, cada bloque ocupa 100% viewport en mobile
- **Grid de 12 columnas con márgenes amplios**: Contenido respira; no hay sensación de aglomeración

#### Signature Elements
1. **Línea divisoria técnica**: Línea diagonal fina (1px) que separa secciones, rotada 15°
2. **Iconografía minimalista**: Líneas simples (stroke-only) para servicios; sin rellenos
3. **Tipografía de contraste**: Serif para títulos (Playfair Display), sans-serif para cuerpo (Roboto)

#### Interaction Philosophy
- **Hover effects sutiles**: Cambio de color verde + ligera elevación (shadow)
- **Botones sin bordes**: Solo fondo verde + texto blanco; al hover, fondo se oscurece
- **Scroll reveal**: Elementos aparecen con fade-in suave (200ms) al entrar en viewport

#### Animation
- **Entrada de página**: Fade-in del hero en 400ms (no bounce)
- **Scroll**: Parallax sutil (5-10% offset) en imágenes de fondo
- **Hover en cards de servicios**: Borde izquierdo verde se expande (3px → 6px) en 150ms
- **CTA flotante**: Pulse suave cada 5 segundos (opacity 0.8 → 1) para captar atención sin agresividad

#### Typography System
- **Títulos (H1-H2)**: Playfair Display, 700, 48px (desktop) / 32px (mobile), line-height 1.2
- **Subtítulos (H3)**: Roboto, 600, 24px, line-height 1.4
- **Cuerpo**: Roboto, 400, 16px, line-height 1.6, color gris oscuro
- **Jerarquía**: Tamaño + peso + color; sin decoraciones adicionales

## <response>

### Idea 2: Diseño Orgánico + Energía Ambiental
**Probabilidad: 0.07**

#### Design Movement
**Biophilic Design + Ilustración Contemporánea**

#### Core Principles
1. **Conexión con naturaleza**: Formas curvas, colores naturales, ilustraciones que celebran la biodiversidad
2. **Narrativa emocional**: No vendemos "muerte de plagas"; vendemos "protección del hogar"
3. **Movimiento fluido**: Transiciones suaves, animaciones que imitan movimiento orgánico
4. **Accesibilidad visual**: Colores cálidos, tipografía legible, suficiente contraste

#### Color Philosophy
- **Paleta**: Verde menta (#a8d5ba) + Terracota (#d97757) + Crema (#f5f1e8) + Azul cielo (#4a90a4)
- **Razonamiento**: Menta = frescura/limpieza; Terracota = tierra/naturaleza; Crema = calidez; Azul = confianza
- **Intención emocional**: "Cuidamos tu espacio con respeto por el entorno"

#### Layout Paradigm
- **Secciones con formas orgánicas**: Uso de clip-path para crear divisiones ondulantes entre secciones
- **Asymmetric grid**: Contenido fluye en columnas variables (3-4 columnas en desktop)
- **Espacios negativos generosos**: Mucho aire blanco/crema para que el ojo descanse

#### Signature Elements
1. **Ilustraciones custom**: Plagas dibujadas con estilo amigable (no terrorífico); técnicos en acción
2. **Formas geométricas suaves**: Círculos, óvalos, ondas; evitar ángulos rectos
3. **Textura de fondo sutil**: Patrón de puntos o líneas finas (opacity 5%) en secciones clave

#### Interaction Philosophy
- **Hover en servicios**: Card se eleva con shadow suave + borde menta aparece
- **Botones redondeados**: Border-radius 50px; relleno terracota; al hover, fondo se vuelve más oscuro
- **Micro-interacciones**: Pequeños iconos animados (ej. gota de agua que cae) en hover

#### Animation
- **Entrada de página**: Fade-in + slide-up suave (300ms) para hero
- **Scroll**: Elementos aparecen con parallax orgánico (formas se mueven a velocidades diferentes)
- **Hover en cards**: Escala 1.02 + shadow aumenta + color de fondo cambia a menta clara
- **CTA flotante**: Pulse suave + cambio de color terracota → naranja en loop

#### Typography System
- **Títulos (H1-H2)**: Poppins, 700, 48px (desktop) / 32px (mobile), line-height 1.3
- **Subtítulos (H3)**: Poppins, 600, 24px, line-height 1.5
- **Cuerpo**: Open Sans, 400, 16px, line-height 1.7, color gris cálido
- **Jerarquía**: Tamaño + color (terracota para énfasis) + peso

## <response>

### Idea 3: Modernismo Dinámico + Urgencia Estratégica
**Probabilidad: 0.06**

#### Design Movement
**Neobrutalism + Diseño de Urgencia (inspirado en startups tech)**

#### Core Principles
1. **Velocidad visual**: Elementos grandes, tipografía bold, colores vibrantes
2. **Contraste extremo**: Fondos oscuros + textos brillantes = atención inmediata
3. **Modularidad agresiva**: Cards, bloques, secciones claramente delimitadas
4. **Energía controlada**: Movimiento constante pero no caótico

#### Color Philosophy
- **Paleta**: Negro (#0a0a0a) + Naranja vibrante (#ff6b35) + Amarillo limón (#f7ff00) + Gris claro (#e8e8e8)
- **Razonamiento**: Naranja = urgencia/acción; Amarillo = alerta; Negro = sofisticación; contraste extremo = conversión
- **Intención emocional**: "Actúa ahora; no esperes a que la plaga llegue"

#### Layout Paradigm
- **Hero full-width con imagen oscura**: Texto blanco/amarillo superpuesto con alto contraste
- **Grid de cards de 2-3 columnas**: Cada servicio en card con borde naranja grueso (3px)
- **Secciones con fondo alternado**: Negro → Gris claro → Negro, crea ritmo visual

#### Signature Elements
1. **Bordes gruesos de naranja**: Líneas de 3-4px que enmarcan secciones y cards
2. **Tipografía sans-serif bold**: Montserrat o Bebas Nova para máximo impacto
3. **Iconografía geométrica**: Formas simples, rellenas, en naranja/amarillo

#### Interaction Philosophy
- **Hover en cards**: Borde se expande (3px → 6px) + fondo cambia a naranja oscuro
- **Botones agresivos**: Fondo naranja + texto negro; al hover, fondo amarillo + texto negro
- **Feedback inmediato**: Todos los clicks producen feedback visual (color change, scale)

#### Animation
- **Entrada de página**: Fade-in rápido (200ms) + slide-down del hero
- **Scroll**: Fade-in de cards con stagger (cada card entra 100ms después)
- **Hover en cards**: Escala 1.05 + borde se anima + shadow aumenta
- **CTA flotante**: Pulsa constantemente (opacity 0.7 → 1) con color naranja → amarillo

#### Typography System
- **Títulos (H1-H2)**: Bebas Nova, 700, 52px (desktop) / 36px (mobile), line-height 1.1
- **Subtítulos (H3)**: Montserrat, 700, 28px, line-height 1.3
- **Cuerpo**: Roboto, 400, 16px, line-height 1.6, color gris claro
- **Jerarquía**: Tamaño + peso + color (naranja/amarillo para énfasis)

---

## Decisión Final

**Seleccionado: Idea 1 - Minimalismo Técnico + Autoridad Profesional**

### Justificación
- **CRO**: Estructura clara y jerárquica maximiza conversiones (menos distracciones)
- **Confianza**: Verde + Gris + Serif transmiten profesionalismo técnico (crítico para plagas)
- **Mobile-First**: Diseño asimétrico se adapta perfectamente a móvil
- **SEO**: Estructura limpia facilita indexación y schema markup
- **Diferenciación**: Evita el cliché de diseños "amigables" que abundan en el sector

### Estilo Visual Definitivo
- **Colores**: Gris oscuro (#1a1a1a), Verde forestal (#1d4620), Blanco puro
- **Tipografía**: Playfair Display (títulos) + Roboto (cuerpo)
- **Espaciado**: Márgenes amplios, grid de 12 columnas
- **Movimiento**: Transiciones suaves, parallax sutil, hover effects minimalistas
- **Elementos**: Líneas diagonales, iconografía de trazo, cards con borde izquierdo verde
